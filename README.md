# Complete Glassware Guide

A comprehensive reference for every type of drink glass, their uses, capacities, and how to choose the right glassware for any beverage.

## Quick Selection Guide

| Drink | Recommended Glass | Why |
|-------|------------------|-----|
| Old Fashioned | Rocks glass | Accommodates ice, spirit-forward |
| Martini | Martini glass or Coupe | Elegant, no ice |
| Margarita | Rocks or Coupe | Salt rim, over ice or up |
| Whiskey tasting | Glencairn | Concentrates aromas |
| Wine | Wine glass (varietal-specific) | Enhances aromatics |
| Beer | Depends on style | See beer glass guide |
| Champagne | Flute or Coupe | Preserves bubbles |
| Highball drinks | Highball glass | Tall, room for mixer |
